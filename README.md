|ROAS      | ≥2.0          | 重组受众结构+调整出价策略          | 12小时内 |

### 3.2 智能决策模型
```python
def optimization_logic():
    if 黄金时段转化率激增:
        自动追加30%预算

        启动实验素材投放

        维持当前投放节奏
